TexTransCore
================================================================================

Location: third-party/TexTransCore/

Description:
  TexTransCore is a .NET library for texture processing in VRM models. It provides
  high-performance texture manipulation capabilities without Unity dependencies.
  TexTransCore is the generic C# core component of the TexTransTool project,
  designed to be reusable across different platforms and applications.

  Repository: https://github.com/ReinaS-64892/TexTransCore
